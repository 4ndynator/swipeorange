# Issue 04: Variants, modifiers, and pricing adjustments

## Goal
Support configurable items with variant and modifier groups.

## Scope
- Variant groups/options with price delta
- Modifier groups/options with optional/required rules and price delta
- Display in item editor and customer detail

## Acceptance Criteria
- Admin can manage variant/modifier groups
- Customer detail view shows full configuration options
- Price adjustments are clearly visible
