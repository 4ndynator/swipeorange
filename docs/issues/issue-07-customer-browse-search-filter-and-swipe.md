# Issue 07: Customer browsing, search, filter, and swipe UX

## Goal
Deliver a swipe-friendly customer browsing experience.

## Scope
- Swipeable item cards
- Tap controls as fallback navigation
- Category filter
- Search by item name
- Detail screen with image, pricing, variants/modifiers, badges

## Acceptance Criteria
- Guest can swipe and browse items smoothly
- Search/filter works client-side
- Detail view shows complete item information
