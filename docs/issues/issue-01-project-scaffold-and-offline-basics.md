# Issue 01: Project scaffold and offline basics

## Goal
Create a mobile-first local-only web app foundation with React + TypeScript and offline-ready static asset behavior.

## Scope
- Replace starter UI with app shell
- Base responsive layout
- Register basic service-worker-compatible structure (offline-ready app shell behavior)
- App-level navigation between Home, Admin, and Customer modes

## Acceptance Criteria
- App runs locally with `npm run dev`
- Build succeeds with `npm run build`
- Layout is responsive and touch-friendly
- User can switch modes from the app shell
