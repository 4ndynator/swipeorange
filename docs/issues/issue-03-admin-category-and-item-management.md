# Issue 03: Admin category and item management

## Goal
Enable local management of categories and menu items.

## Scope
- CRUD categories and items
- Reorder categories and items
- Duplicate items
- Visibility/availability controls
- Delete warning for non-empty categories

## Acceptance Criteria
- User can create/edit/delete/reorder categories and items
- Duplicate item action works
- Hidden items are excluded from customer mode
- Delete confirmation/warning is shown where required
